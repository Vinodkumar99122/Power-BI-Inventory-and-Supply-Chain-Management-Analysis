# Power-BI-Inventory-and-Supply-Chain-Management-Analysis
## Project Overview
- This project presents an end-to-end Inventory & Supply Chain Analytics Dashboard built using Power BI.
- The objective is to analyze inventory performance, demand patterns, supplier efficiency, and operational KPIs to support data-driven decision-making.
- This dashboard simulates real-world supply chain scenarios and provides actionable business insights.

## Business Problem
- Excess inventory holding costs
- Stockout risks
- Supplier delays
- Poor demand visibility
- Inefficient reorder planning
- This project addresses these challenges using data visualization and KPI tracking.

## Key Insights & Analysis
> Fast-moving vs Slow-moving product analysis
> Inventory turnover tracking
> Reorder level monitoring
> Supplier performance evaluation
> Delivery lead time analysis
> Stock availability and backorder insights

## KPIs Designed
- Total Inventory Value
- Stock Turnover Ratio
- Average Lead Time
- Reorder Point Monitoring
- Supplier Performance Score
- Stockout Risk Indicator
